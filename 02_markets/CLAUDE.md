# 02_markets — 市場展開

## 展開ステータス

| 市場 | 状態 | 特記事項 |
|---|---|---|
| 🇸🇬 Singapore | ✅ **Active** | 英語・中国語圏。酒類規制は比較的整備。高所得・プレミアム親和性高い。 |
| 🇲🇾 Malaysia | 🔜 **Next** | 多民族（マレー系・中国系・インド系）。ハラール考慮必須。非ムスリム向け市場は存在。 |
| 🇮🇩 Indonesia | 🔜 **Upcoming** | 世界最大のムスリム国家。酒類は非常に厳しい規制。富裕層・観光地限定戦略が現実的。 |
| 🇭🇰 Hong Kong | 🔜 **Upcoming** | 中国語圏。日本酒・ウイスキー文化が根付いている。高所得・プレミアム市場。 |
| 🇮🇳 India | 🔜 **Long-term** | 州ごとに酒類規制が大きく異なる。都市部富裕層向け。成長市場だが複雑。 |

## 各市場フォルダ構成
```
{market}/
├── CLAUDE.md        — 市場固有のコンテキスト
├── overview.md      — 市場概況・KPI
├── regulations/     — 酒類販売ライセンス・輸入規制
├── operations/      — チャネル別オペレーション
│   ├── ec/
│   ├── b2b/
│   └── restaurant-bar/
├── partnerships/    — 現地パートナー・ディストリビューター
└── entry-strategy.md — 進出戦略（未進出市場）
```

## 規制対応の優先度
酒類規制は市場参入の最大リスク。各市場フォルダの `regulations/` を必ず確認すること。
法務相談は `07_legal/` と連携。
