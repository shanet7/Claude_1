# 05_marketing — マーケティング

## ブランドトーン・オブ・ボイス

| 要素 | 内容 |
|---|---|
| **トーン** | 知的・温かい・本物志向。教えるが押しつけない。 |
| **言語スタイル** | 簡潔・詩的。説明より体験を想起させる。 |
| **NG** | 過度な敬語・堅すぎる表現・観光ガイド的な説明口調 |
| **OK** | 「この一杯が、日本の山里を連れてくる」的な表現 |

## チャネル戦略

| チャネル | 目的 | KPI |
|---|---|---|
| Instagram | ブランド認知・ビジュアル | フォロワー数・エンゲージメント率 |
| Facebook | コミュニティ・イベント告知 | リーチ・イベント参加 |
| TikTok | 若年層獲得・文化教育 | 視聴数・フォロワー |
| Email | 顧客育成・購買促進 | 開封率・CVR |
| Blog/コンテンツ | SEO・専門性訴求 | オーガニック流入 |

## フォルダ構成
- `strategy/` — マーケティング戦略・年間計画
- `content/social-media/` — SNS投稿（プラットフォーム別）
- `content/blog/` — ブログ記事・コラム
- `content/email/` — メールマガジン・ニュースレター
- `campaigns/` — キャンペーン企画・実績
- `events/` — イベント企画・報告
- `pr/` — プレスリリース・メディア対応
