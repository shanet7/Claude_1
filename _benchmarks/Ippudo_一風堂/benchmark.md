# 一風堂（Ippudo / DWS） — ベンチマーク分析

## 企業概要
- **設立:** 1985年（博多）/ 運営: 力の源ホールディングス
- **業態:** ラーメン（F&B）の直営・フランチャイズ展開
- **海外展開:** 15カ国以上、100店舗超
- **アジア:** SG・HK・台湾・韓国・豪州等に展開

## UCHIが参照すべき点（F&B展開）

### 1. SEA参入モデル
- **シンガポールをSEAハブ**として確立し、そこから展開
- SG→MY→HK→豪州 の順で展開（UCHIと類似）
- 現地法人設立 + 現地パートナー（JV）の組み合わせ

### 2. 「本物」を維持しながらスケールする仕組み
- セントラルキッチン方式でスープの品質を管理
- スタッフ研修は日本本部で行い、現地に展開
→ UCHI応用: 日本酒の品質管理 + スタッフの利き酒教育を日本で実施

### 3. ブランドアンバサダーとしてのスタッフ
- 「博多の文化」を語れるスタッフを育てる
- ホスピタリティ × 食文化知識の両立
→ UCHI応用: スタッフが「日本酒文化の語り手」になるトレーニング体系

### 4. プレミアム × アクセシブルのバランス
- ラーメンという「庶民の食」をプレミアム化
- SG/HKでは客単価S$20-30に設定（現地プレミアム）
→ UCHI応用: 日本酒を「本格的だが近づきやすい」ポジションに

### 5. イベント・期間限定メニュー
- 蔵元コラボ、地域限定メニューで話題性を作る
→ UCHI応用: 季節の蔵元コラボイベント、利き酒セッション

## 組織図（参考）
```
力の源ホールディングス
├── 国内事業部（直営・FC）
└── 海外事業部
    ├── シンガポール法人（SEAハブ）
    ├── 香港法人
    └── その他地域法人
```

## UCHIへの最重要示唆
> 「フランチャイズではなく直営を守ることで、ブランド体験の品質を担保した」

→ UCHIのRestaurant-Barは直営維持が原則。FC展開は慎重に。
