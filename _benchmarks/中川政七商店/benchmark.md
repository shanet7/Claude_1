# 中川政七商店 — ベンチマーク分析

## 企業概要
- **設立:** 1716年（奈良）
- **業態:** 日本工芸・生活雑貨の企画・製造・小売
- **ミッション:** 「日本の工芸を元気にする」
- **売上:** 約100億円規模（2020年代）
- **店舗数:** 50店舗以上（全国）+ 海外は限定的

## UCHIが参照すべき点

### 1. ミッションドリブン経営
「商品を売る前に、文化を守るミッションを語る」
→ UCHI応用: 「日本の造り手を守り、アジアに届ける」という大義

### 2. ブランドストーリーテリング
- 産地・職人・製法の物語を前面に出す
- 商品説明より「なぜこの商品が存在するか」を語る
→ UCHI応用: 各蔵元のストーリーを前面に（Dassaiが「山口の山奥で」という話をするように）

### 3. 編集力（キュレーション）
全国6,000以上の工芸メーカーから厳選した商品のみ扱う
→ UCHI応用: 数百ある日本酒蔵から「UCHIが選ぶ理由がある蔵」のみに絞る

### 4. 「遊中川」「粋更」等のマルチブランド展開
価格帯・ターゲットに応じたブランド分け
→ UCHI応用: プレミアムライン / スタンダードライン の設計

### 5. 組織設計
- **事業本部制:** 企画・製造・小売が一体
- **フラット組織:** 若手権限委譲
- **社員が語り手:** スタッフ全員がブランドアンバサダー

## 組織図（参考）
```
代表取締役
├── 事業本部
│   ├── 商品部（企画・開発）
│   ├── 販売部（EC・店舗）
│   └── プロデュース部（他社支援）
├── マーケティング本部
└── 管理本部（人事・財務・法務）
```

## UCHIが学ぶべき優先ポイント
1. ブランドコミュニケーション（言語・ビジュアル）
2. 産地・造り手との関係構築モデル
3. スタッフをブランドの語り手として育成する方法
