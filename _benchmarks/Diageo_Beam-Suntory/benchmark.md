# Diageo / Beam Suntory — ベンチマーク分析

## 企業概要
- **Diageo:** 英国。世界最大のアルコール企業。Johnnie Walker, Guinness, Tanqueray等
- **Beam Suntory:** 日本。Suntory × Jim Beam合弁。Yamazaki, Hakushu, Toki等
- **アジア売上:** 両社ともアジアが成長エンジン

> UCHIの競合ではなく、**業界の規制・流通・マーケティングモデルの参照先**

## UCHIが参照すべき点（アルコール業界のグローバル展開）

### 1. 国別ライセンス・規制対応フレームワーク
- **各国で輸入業者ライセンス + 販売ライセンスを個別取得**
- 現地法律事務所との継続的パートナーシップ
- 規制モニタリング体制の整備
→ UCHI応用: `07_legal/licenses/` に国別ロードマップを整備

### 2. ディストリビューターネットワーク管理
- 各市場で「マスターディストリビューター」+ 「サブディストリビューター」の2層構造
- ディストリビューターへの教育・サポートプログラム
- 売上・在庫の可視化（システム連携）
→ UCHI応用（中小規模での応用）: パートナー選定基準の明確化

### 3. On-Premise（バー・レストラン）戦略
- Diageo の "Bar Academy" : バーテンダー教育プログラム
- 店舗での試飲・体験を通じた消費者教育
→ UCHI応用: B2Bクライアント（バー・レストラン）への「スタッフ教育プログラム」提供

### 4. ポートフォリオ管理
- 価格帯別（Entry / Mid / Premium / Ultra Premium）で商品ラインを設計
→ UCHI応用: 日本酒・焼酎・ウイスキーの価格帯設計

### 5. Responsible Drinking（責任ある飲酒）
- 各市場の規制に合わせた「責任ある飲酒」メッセージ
- 特にイスラム圏・厳格市場ではコンプライアンスが最重要
→ UCHI応用: インドネシア・マレーシアでのコミュニケーション設計
