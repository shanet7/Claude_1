# 獺祭（Dassai）/ 旭酒造 — ベンチマーク分析

## 企業概要
- **蔵元:** 旭酒造（山口県岩国市）
- **創業:** 1948年、獺祭ブランドは1990年代に確立
- **売上:** 約140億円（2023年）/ 輸出比率: 約20-25%
- **主要輸出市場:** 米国・フランス・香港・シンガポール・台湾

## UCHIが参照すべき点（日本酒ブランド国際展開）

### 1. ストーリーの一点突破
「山奥の小さな蔵が、山田錦にこだわり世界へ」
→ 弱点（山奥・小規模）をブランドストーリーに転換
→ **UCHI応用:** 各蔵元の「弱点」が「物語」になる

### 2. 直接輸出による品質・価格コントロール
- ディストリビューターに頼りすぎず、直接輸出で価格と品質を管理
- 現地インポーターとの契約で「ブランドガイドライン」を厳守させる
→ UCHI応用: ディストリビューター選定時の「ブランド保護条項」

### 3. プレミアムポジションの維持
- 安売り禁止、値引き禁止のポリシーを徹底
- 希少性を武器にする（年間製造量の管理）
→ UCHI応用: B2Bクライアントへの値引き要求への対応方針

### 4. 教育 × 体験で市場を作る
- ニューヨーク旗艦店「Dassai Blue」開設（2023）
- 現地でのイベント・試飲会で文化普及
→ UCHI応用: Restaurant-Barがまさにこの役割

### 5. パリとのコラボ（Joel Robuchon）
世界的シェフとのコラボで「日本酒 × 世界の食文化」を演出
→ UCHI応用: SG/HKの著名シェフとのコラボイベント

## 旭酒造の組織（参考）

```
代表取締役（桜井会長 / 桜井社長）
├── 製造部門
├── 国内営業部門
├── 海外部門
│   ├── 米国法人
│   └── 各国代理店管理
└── マーケティング・PR
```

## UCHIへの最重要示唆
> 「プレミアム日本酒は、味だけでは売れない。教育と体験が市場をつくる」

→ UCHIが扱う蔵元は、蔵元自身の国際展開戦略と連携することを検討。
→ 蔵元を「共同マーケティングパートナー」として位置づける交渉が有効。

## 仕入れ関係での参考
- 獺祭は独占エージェントを嫌う → 非独占で複数ディストリビューター
- UCHIはむしろ「文化を語れる専門家」として差別化できる
