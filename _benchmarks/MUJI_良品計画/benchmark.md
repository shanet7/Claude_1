# MUJI（良品計画）— ベンチマーク分析

## 企業概要
- **設立:** 1980年（西友のPBとして）、1989年良品計画設立
- **業態:** ライフスタイル雑貨・衣料・食品の製造小売
- **ミッション:** 「感じ良いくらし」
- **海外展開:** 30カ国以上、1,000店舗超
- **アジア売上比率:** 約40%（中国・アジアが主力）

## UCHIが参照すべき点

### 1. アジア展開フレームワーク（最重要）
MUJIのアジア展開ロードマップ:
```
日本で確立 → 香港・台湾（中華圏） → SG（ハブ） → MY/ID/TH → IN（長期）
```
→ **UCHI応用:** SG確立 → HK（文化親和性） → MY → ID/IN の順序はMUJIに近い

### 2. 標準化 × 現地化の判断軸
- **標準化:** ブランドトーン・価格帯・商品品質
- **現地化:** 店舗サイズ・一部商品・コミュニケーション
→ UCHI応用: メニュー・コンテンツは現地語対応、ブランド体験は統一

### 3. フラッグシップ戦略
各市場で最初はプレミアム立地に「旗艦店」を出す
→ UCHI応用: 各市場の直営店（Restaurant-Bar）はブランド確立のための投資

### 4. サプライチェーン管理
素材・産地の透明性をブランド価値に転換
→ UCHI応用: 蔵元・原料・製造プロセスの透明性（トレーサビリティ）

### 5. ECと店舗のシナジー（MUJI passport）
会員制でEC・店舗を統合、顧客データを一元管理
→ UCHI応用: 3チャネルを横断する会員プログラムの設計

## 組織図（参考）
```
CEO
├── 国内事業本部
│   ├── 商品部
│   ├── 販売部
│   └── EC事業部
├── 海外事業本部（地域別）
│   ├── 東アジア（中国・香港・台湾・韓国）
│   ├── 東南アジア（SG・MY・ID・TH等）
│   └── 欧米（UK・US等）
├── 商品開発本部
└── 経営管理本部
```

## UCHIへの最重要示唆
> 「MUJIは進出する市場で最初から利益を狙わない。まず『MUJIとは何か』を伝えることに投資する」

→ Restaurant-Barを「マーケティング費用」と割り切るMUJI的思考が有効。
