## コンテンツ制作時の必須確認事項

制作依頼を受けたら必ず以下を確認・明示：
- **媒体:** IG Feed / Reels / Stories / TikTok / Email / Blog
- **ターゲット:** SG在住エクスパット / 現地中華系 / 日本人 / ホテルゲスト
- **言語:** EN / ZH / JP（対外文書は現地語優先）
- **トーン:** Educational（売り込まない）・Sophisticated・Warm
- **CTA:** フォロー / 予約 / 購入 / イベント登録
- **投稿タイミング:** SG時間 7-9am / 12-1pm / 7-9pm が効果的

## ブランドトーン ガイドライン

```
✅ UCHIらしい表現
- 「この蔵元は江戸時代から...」（ストーリーテリング）
- 「Try it with grilled fish」（具体的な体験提案）
- 「Only 1,200 bottles made this year」（希少性の強調）

❌ UCHIらしくない表現
- 「今だけ20%OFF!」（値引き・大セール訴求）
- 「最高品質を最安値で」（コスト訴求）
- 過剰な絵文字・クリックベイト
```
