# 10_projects — プロジェクト管理

## フォルダ構成
- `active/` — 進行中プロジェクト
- `pipeline/` — 検討中・計画中
- `archive/` — 完了済み

## プロジェクトファイルの命名規則

```
YYYYMM_[市場コード]_[チャネルコード]_[プロジェクト名].md

例:
202603_MY_B2B_ディストリビューター選定.md
202604_SG_BAR_Spring-Sake-Event.md
202606_HK_ALL_市場参入計画.md
```

## 市場コード
- SG / MY / ID / HK / IN / ALL

## チャネルコード
- EC / B2B / BAR / ALL

## プロジェクトテンプレート
各プロジェクトファイルの標準構成:
```
# プロジェクト名
## 目的
## 担当・関係者
## タイムライン
## 予算
## 成功指標（KPI）
## 進捗・メモ
## 決定事項・ログ
```
