# URA Pre-application Consultation — 準備ガイド

**目的:** カフェ営業（Food Shop）追加に際し、Change of Use（用途変更）が必要かどうかをURAに確認する。

---

## 予約方法

1. GoBusiness にログイン: https://www.gobusiness.gov.sg
2. 「Licences & Permits」→「Pre-application Guidance」→「URA」を選択
3. 「Development Control」の相談を予約（オンラインまたは対面）
4. **予約時に持参・添付するもの:**
   - 現在の店舗の賃貸契約書（または所有権証明）
   - 現在の用途（Approved Use）が分かる書類
   - 建物の住所・階・ユニット番号

---

## URAに聞くべき質問リスト（この通りに質問する）

```
1. "Our current premises is approved as a 'Restaurant/Bar'. 
   We plan to add daytime café operations (11am-5pm) serving 
   non-alcoholic beverages and desserts only. 
   Is a Change of Use application required?"

2. "If Change of Use is required, what is the typical processing 
   time and cost?"

3. "Are there any restrictions on the type of food/beverage 
   operations in this zoning area?"

4. "We will be using existing kitchen facilities. 
   Is a new kitchen fitting-out plan submission required?"
```

---

## 持参するもの（当日）

- [ ] 現在のTenancy Agreement（賃貸契約書）
- [ ] 既存のFood Shop Licence（またはLiquor Licence）のコピー
- [ ] 店舗フロアプランのスケッチ（手書きでも可）
- [ ] このチェックリストを印刷

---

## 回答後にClaude（私）に共有してほしいこと

- Change of Use が必要か → Yes / No
- 必要な場合の処理期間
- その他指摘事項
