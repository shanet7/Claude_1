# 04_products — 商品・プロダクト管理

## カテゴリ構成

| カテゴリ | フォルダ | 説明 |
|---|---|---|
| 日本酒 | `sake/` | 純米・吟醸・大吟醸・スパークリング等 |
| ウイスキー | `whisky/` | Suntory, Nikka, 地ウイスキー等 |
| 焼酎 | `shochu/` | 芋・麦・米・泡盛 |
| ビール | `beer/` | クラフトビール・地ビール |
| ノンアルコール | `non-alcoholic/` | 甘酒・梅酒ノンアル・茶割り等 |
| 仕入れ | `sourcing/` | 蔵元・サプライヤー管理 |

## 商品管理の基本方針
1. **厳選主義:** カタログを広げすぎない（獺祭モデル参照）
2. **ストーリー必須:** 各商品に蔵元・製法・ペアリングのストーリーを記載
3. **3チャネル対応:** EC用・B2B用・レストランバー用の説明文を別管理
4. **多言語化:** EN（基本）+ ZH（HK/SG向け）+ BM（MY向け）
