# 03_channels — 販売チャネル戦略

## 3チャネルの位置づけ

```
                    ┌─────────────────────────────────────┐
                    │         UCHI Channel Strategy        │
                    └─────────────────────────────────────┘

  Restaurant/Bar              B2B                    EC
  ──────────────         ──────────────        ──────────────
  ブランド体験           量・安定収益           継続・データ
  文化教育               ホテル/レストラン       個人顧客
  コミュニティ           リテール卸              定期購入
       ↓                      ↓                     ↑
  ファン獲得 ────────────────────────────────→ リピート購入
```

## シナジー設計
- **Restaurant-Bar → EC:** 体験→購入の導線（QRコード、会員登録）
- **B2B → EC:** ビジネスバイヤーの個人購買転換
- **EC → Restaurant-Bar:** オンライン限定イベント招待
